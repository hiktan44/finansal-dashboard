# TÜİK TÜFE Verileri Toplama Planı

## Görev Tanımı
TÜİK web sitesinden (www.tuik.gov.tr) TÜFE (Tüketici Fiyat Endeksi) enflasyon verilerini toplamak. Kasım 2025'e kadar olan tüm aylık TÜFE verilerini bulmak.

## Hedef Veri İçeriği
- Aylık TÜFE oranları
- Yıllıklandırılmış oranlar  
- Alt kalem detayları
- Excel veya PDF formatında veri dosyaları
