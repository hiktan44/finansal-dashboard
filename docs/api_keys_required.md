# API Anahtarı Gerekli

Gerçek veri toplama işlemini tamamlamak için aşağıdaki kurumlardan API anahtarları gerekmektedir:

## Gerekli API Anahtarları:

### 1. FRED API (Federal Reserve)
- Website: https://fred.stlouisfed.org/docs/api/fred.html
- Ücretsiz kayıt ile alınabilir
- Kullanılacak veriler: Federal Funds Rate, Unemployment Rate, GDP

### 2. ECB API (European Central Bank) 
- Website: https://data.ecb.europa.eu/help/api/overview
- Kullanılacak veriler: ECB Interest Rates, Eurozone Inflation, GDP

### 3. IMF API (International Monetary Fund)
- Website: https://data.imf.org/en/Resource-Pages/IMF-API
- Kullanılacak veriler: Global GDP Growth, WEO Data, Exchange Rates

### 4. OECD API (Organisation for Economic Co-operation and Development)
- Website: https://data-explorer.oecd.org/
- Kullanılacak veriler: Economic Indicators, Industrial Production, CPI
